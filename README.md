Esse Projeto tem como o objetivo, fornecer um sistema de Banco de dados para a gerência de uma locadora de equipamentos de construção. 
Esse sistema serve para maior facilidade na gerência dos dados e recursos da empresa, servindo para o armazenamento de todos os dados de locação, 
transporte, cliente, funcionário, contrato e equipamentos em que a empresa possui. Deixando-os relacionados, para que possam se comunicar entre si.
